## 스터디 규칙
1. 매주 화요일 20:30 에 스터디 진행
2. 매주 2팀씩 번갈아가면서 3문제씩 출제한다.
3. 출제는 백준, 프로그래머스 상관 없으며, 백준 그룹에 등록한다
4. 주차별 본인 폴더에 소스코드 업로드한다.   
5. **문제풀이는 스터디 하는 날 00:00까지 깃에 업로드 한 경우만 인정된다.**
