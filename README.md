## Git 사용 규칙
1. [주차]/GITNAME으로 브랜치를 생성한다. -> [5주차]Kodo92, [6주차]Kodo92
2. 주차별로 브랜치를 새로 생성한다.
3. 커밋은 1문제씩 커밋하며, 커밋 내용은 아래 규칙을 적용한다.
   - 사용한 알고리즘 - 문제이름
   - 예시 : DP - RGB
4. 해당 주차의 문제를 다 풀었다면 PR을 날린다. PR은 그 다음 차수 스터디 시작할 때 일괄적으로 Merge한다.
5. PR에서 코드리뷰를 한다.
